Sobre o projeto

Aplicação web responsiva pronta para GitHub Pages e Google Apps Script.

  • Login com Google OAuth
  • Separação total dos dados por usuário
  • PWA instalável em Android e iPhone
  • Modo offline para consulta
  • Exportação em CSV e Excel
  • Dashboard e relatórios com Chart.js
